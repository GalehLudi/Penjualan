## Installation
1. Clone Repo
    ```console
    git clone https://github.com/GalehLudi/Penjualan.git
    ```

2. Run
   ```console
   php artisan key:generate
   ```
   ```console
   php artisan migrate
   or
   php artisan migrate:refresh
   ```
   ```console
   php artisan db:seed
   ```

3. Run server
   ```console
   php artisan serve
   ```
